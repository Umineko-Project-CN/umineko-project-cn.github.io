## 教程

> 更多帮助请参照 [帮助页面](/help#ps3-版移植版)。

### 配置要求

**最低配置要求：**

- **处理器：**3.0 GHz 单核处理器（需支持 SSE3 指令集）
- **显卡：**兼容 OpenGL 2.1 或 DirectX 9；拥有 256 MB 显存且纹理尺寸限制至少为 4096 px
- **内存：**2 GB（32 位 Windows 平台可能会出现问题）
- **储存空间：**15 GB 可用空间
- **显示分辨率：**1280×720 或更高
- **操作系统：**macOS 10.6 或更高；支持兼容库的 Linux 64位；Windows XP SP3 或更高；iOS 8.0 或更高；Android 4.1 或更高

推荐使用更新的操作系统版本。

### 安装步骤

#### Windows 平台

**下载：**

1. 从“下载链接”的“**游戏下载**”中，下载“**1. 完整游戏**”文件夹中的所有压缩文件。

**解压：**

1. 将全部压缩文件下载至同一目录下，使用**解压密码**，直接解压其中**第一个分包**即可得到完整游戏。
    - 切勿修改文件后缀！
    - 推荐使用 [7-Zip](https://www.7-zip.org/download.html)、[PeaZip](https://peazip.github.io/)、[WinRAR](https://www.rarlab.com/download.htm) 等常用压缩软件。
2. 若解压出错，通常是软件版本过旧或下载损坏。请使用推荐压缩软件，并核对分包的 **SHA1 校验码**（在“1. 完整游戏”文件夹内的 `SHA1.txt` 中查看）。若不一致需重新下载该分包。

**更新与启动：**

![](@assets/pages/download/port/download_read2.webp)

1. 使用文件夹内的 **“海猫鸣泣之时.exe”（启动器）** 启动，以保证版本最新。
    - 若启动器被杀毒软件误删，可前往网盘“3. 启动器”文件夹重新下载。
    - 也可从“下载链接”的“**更新下载**”中，分别下载“脚本”与“资源包”，自行解压到游戏目录。
2. 启动器运行后会自动联网检查更新：
    - 指示灯为 **红色或橙色：** 请等待“获取更新”按钮出现并点击更新。
    - 指示灯为 **绿色：** 版本已是最新，可直接点击“开始游戏”。
3. 若遇重大版本更新，可能会强制跳转至网盘手动下载更新。
4. Windows 7 用户若更新失败，请下载安装 [.NET Framework 4.8](https://dotnet.microsoft.com/download/dotnet-framework/thank-you/net48-offline-installer)。

### 解压密码

拥有正版 PS3 版与 PC 版时，将下述四个部分按顺序拼接，即可得到 27 位解压密码：

1. 找到 PS3 版游戏手册封底上的粗体数字，不带连字符地将其抄下；
2. 查看 PS3 版游戏手册第 4 页底部“KB”前面的数字，把这串数字写在上一步得到的字符串后方；
3. 找到 PC 版“うみねこのなく頃に.exe”文件，并将其“大小”一栏的数字（并非占用空间）添加到字符串后方；
4. 找到 PC 版“nscript.dat”文件，并将其“大小”一栏的数字（并非占用空间）添加到字符串后方。

为方便国内玩家，**下载链接内已随附解压密码**。但根据制作组 Umineko Project 声明，拥有正版仍是游玩必要条件。
